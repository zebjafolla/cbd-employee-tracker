# CBD Installer Tracker

A personal iPhone-friendly work tracker for install, service, and shop pay periods.

## Features
- Wednesday-to-Tuesday repeating pay periods.
- 1-week and 2-week views.
- Install commission tracking with effective hourly rate.
- Install floor checking at $30/hr pre-40 and $45/hr post-40.
- Common hourly rate for non-install work with 1.5x overtime after 40 hours.
- Drive time tracked separately and counted in total hours.
- CSV export for current 2-week period.
- Full backup JSON and settings-only JSON export.
